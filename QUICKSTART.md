# Быстрый старт

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

**Внимание на Windows:** Если возникнут проблемы с установкой `better-sqlite3`, установите инструменты сборки:
- Загрузите [Visual Studio Build Tools](https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2022)
- Выберите "Desktop development with C++" при установке

### 2. Запуск сервера разработки

```bash
npm run dev
```

### 3. Откройте браузер

Перейдите на http://localhost:3000

База данных будет автоматически создана в папке `data/sarancha.db` при первом запуске.

## Добавление данных

### Добавить нашествие через SQL

Откройте базу данных `data/sarancha.db` в SQLite браузере и выполните:

```sql
INSERT INTO invasions (year, title, description, full_text, region)
VALUES (2021, 'Название нашествия', 'Краткое описание', 'Полный текст описания...', 'Название региона');
```

### Добавить препарат

```sql
INSERT INTO preparations (name, description, active_substance, application_method)
VALUES ('Название препарата', 'Описание', 'Действующее вещество', 'Способ применения');
```

## Структура файлов

- `/pages` - все страницы сайта
- `/components` - переиспользуемые компоненты
- `/lib` - утилиты и работа с базой данных
- `/locales` - файлы переводов
- `/styles` - стили CSS
- `/public` - статические файлы (изображения и т.д.)
- `/data` - база данных SQLite (создается автоматически)

## Возможные проблемы

### Ошибка при установке better-sqlite3

**Решение:** Установите инструменты сборки (см. выше) или используйте альтернативу:
```bash
npm install --build-from-source
```

### База данных не создается

**Решение:** Убедитесь, что у приложения есть права на запись в папку проекта.

### Карты не отображаются

**Решение:** Проверьте подключение к интернету (карты загружаются с OpenStreetMap).
