# Инструкция по установке и запуску

## Требования

- Node.js 18.x или выше
- npm или yarn

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Инициализация базы данных произойдет автоматически при первом запуске приложения. База данных будет создана в папке `data/sarancha.db`

3. Запустите сервер разработки:
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Структура проекта

```
/
├── pages/              # Страницы Next.js
│   ├── api/           # API endpoints
│   ├── invasions/     # Страницы нашествий
│   └── ...
├── components/         # React компоненты
├── lib/               # Утилиты и работа с БД
├── locales/           # Файлы переводов (kk, ru)
├── styles/            # CSS модули
├── public/            # Статические файлы
└── data/              # База данных SQLite (создается автоматически)
```

## Языки

Проект поддерживает два языка:
- Русский (ru) - по умолчанию
- Казахский (kk)

Переключение языка доступно в шапке сайта.

## База данных

База данных SQLite находится в папке `data/sarancha.db`.

Таблицы:
- `invasions` - нашествия саранчовых
- `preparations` - препараты против саранчовых
- `map_data` - данные для карт
- `invasion_photos` - фотографии нашествий
- `invasion_links` - ссылки на нашествия

## Производственная сборка

```bash
npm run build
npm start
```

## Дополнительные настройки

### Настройка домена

После покупки домена, обновите файл `next.config.js` с настройками вашего домена.

### Настройка хостинга

При развертывании на хостинге:
1. Убедитесь, что Node.js 18+ установлен
2. Установите зависимости: `npm install --production`
3. Соберите проект: `npm run build`
4. Запустите: `npm start`

Для хостинга с поддержкой статических сайтов (Vercel, Netlify):
- Проект готов к развертыванию через Vercel или Netlify
- База данных SQLite будет работать на серверах с файловой системой

## ГИС сервер

Для полной функциональности карт необходимо развернуть ГИС сервер. В текущей версии используются базовые карты OpenStreetMap через Leaflet.

## QR код

QR код для доступа к сайту можно сгенерировать для любой страницы, добавив параметр `?qr=true` или используя внешние сервисы генерации QR кодов.
